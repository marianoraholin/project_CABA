# Project CABA

```bash
Este proyecto se centra en el análisis de datos de alojamientos en Airbnb en la Ciudad de Buenos Aires, con el objetivo de:

Identificar patrones y tendencias clave en el mercado de alojamientos.
Explorar la dinámica del mercado mediante análisis geoespaciales.
	
```
